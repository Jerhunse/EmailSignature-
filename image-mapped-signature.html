Customization Instructions:

To adjust the clickable areas:

  1. Open this HTML file in a browser
  2. Right-click on the image and select "Inspect Element"
  3. Find the <map> element and adjust the coordinates
  4. Update the href attributes with your actual URLs
  5. Test by clicking on different areas of the image

Coordinate System:

  • rect(x1,y1,x2,y2) - Rectangle from top-left (x1,y1) to bottom-right (x2,y2)
  • circle(x,y,radius) - Circle centered at (x,y) with given radius
  • poly(x1,y1,x2,y2,x3,y3,...) - Polygon with multiple points
